## Examples

```python
import rzhupy

print("Шутка:")
print(rzhupy.joke('ru'))
print("\n")

print("die Geschichte:")
print(rzhupy.story('de'))
print("\n")

print("Poème:")
print(rzhupy.poem('fr'))
print("\n")

print("Афоризм:")
print(rzhupy.aphorism('uk'))
print("\n")

print("Quote:")
print(rzhupy.quote('en'))
print("\n")

print("Pane abbrustolito:")
print(rzhupy.toast('it'))
print("\n")

print("Күй:")
print(rzhupy.status('kk'))
```
